基于PHP的在线学习网站的设计与实现
=============
## 项目描述
本项目采取模块化开发，主体分三个部分，各自基本独立，也就是说需要实现三个网站
	
## iSchool
iSchool是本项目的主站，用来显示所有的教程

## iTool
iTool是本项目中所有代码在线编辑运行的站点，负责代码的演示共享等

## iAdmin
iAdmin是本项目的管理后台，负责所有文章编辑，代码编辑，和网站的所有设置相关的设置等

## 一、在线代码编辑器 iTool  [tool.usta.wiki](http://tool.usta.wiki/)

### 本站主要包含以下四大功能：
**1. 代码在线编辑
2. 代码在线编译、运行
3. 代码分享、协作
4. 代码实时归档**
